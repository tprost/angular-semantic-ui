ui.dimmer

Installation

First, get the file:

  • Google CDN e.g. 
    "//ajax.googleapis.com/ajax/libs/angularjs/X.Y.Z/"
  • NPM e.g. 
    npm install @X.Y.Z
  • Bower e.g. 
    bower install #X.Y.Z
  • code.angularjs.org (discouraged for production use) e.g. 
    "//code.angularjs.org/X.Y.Z/"

where X.Y.Z is the AngularJS version you are running.

Then, include undefined in your HTML:

<script src="path/to/angular.js"></script>
<script src="path/to/"></script>

Finally, load the module in your application by adding it as a dependent module:

angular.module('app', ['ui.dimmer']);

With that you're ready to get started!

Module Components

Type

Name Description

The controller used in dimmable directive. Use it to control the behaviour of the dimmer inside the dimmable.
ui.dimmer.DimmerController

The controller used in dimmer directive. Use it to control the behaviour of the dimmer.
