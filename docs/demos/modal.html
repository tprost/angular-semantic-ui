Inline Modal

You can open an existing modal with its controller.

A modal can be created by simply have the modal exist on the page when Angular bootstraps. 
          
angular.element(document.querySelector('.ui.modal'))
.controller('modal').show();;

On-The-Fly Modal

You can open a modal dynamically using `modalService`

A modal can be created by simply have the modal exist on the page when Angular bootstraps. 
          
modalService.openModal();
