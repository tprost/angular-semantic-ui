ui.modal.ModalController

  1. - type in module ui.modal

The controller for the modal directive.

Methods

  • show();

    Animate the modal in.

    Returns

    Promise

    Promise that resolves when the modal gets closed for whatever reason. The result of the promise will be an object with attribute resolution set to either approved, denied or closed.

  • hide();

    Animate the modal out.

    Returns

    Promise

    Promise that resolves when the modal finishes animating out.

  • approve();

    Hides the modal and resolves the modal as approved.

    Returns

    Promise

    Promise that resolves when the modal finishes animating out.

  • deny();

    Hides the modal and resolves the modal as denied.

    Returns

    Promise

    Promise that resolves when the modal finishes animating out.

  • toggle();

    If the modal is active, call hide(), otherwise call show().
